## Set Up Environment
To setup an `environment`, the follow package is needed be installed.
1. request
2. bs4
3. smtplib
4. email.mine (default)
5. datetime (default)

```
pip3 install request
pip3 install beautifulsoup4
```
Check `python3` and import the email & datatime.
```
python3
import smtplib
import datatime
import email.mime
```
